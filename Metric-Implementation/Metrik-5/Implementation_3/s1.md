# S-1 Specification

```
metric(model1, model2):
   isValidParsedModel(model1)
   isValidParsedModel(model2)

   d1 = normalize(model1)
   d2 = normalize(model2)
   sem   = classSem(d1, d2)
   struc = classStruc(d1, d2)

   result = (1 - 0.5) * sem + 0.5 * struc
   return clampToUnit(result)

   requires:
       isValidParsedModel(model1),
       isValidParsedModel(model2)
       isLoadedWordNet()
       isLoadedNetworkX()
       isLoadedNLTK()

   ensures:
       isValidSimilarity(result)

Input:  Two ParsedModel instances (student and instructor).
Output: A real number in [0.0, 1.0] indicating class-diagram similarity.
```

## Invariants


```
isValidParsedModel(M):
    checks if model adheres to the parser data structure.
    class names are unique within M
    enumeration names are unique within M
    every relation references classes that exist in M
    every attribute belongs to exactly one class in M
    M is an instance of Parser.models.ParsedModel
```


```
isValidDiagram(D):
    D is a Diagram instance
    every c in D.classes is a ClassInfo instance
    every a in c.attributes is an AttributeInfo instance
    every c.name is non-empty string
    every a.name is non-empty string
    every a.type is a string (may be empty)
    every a.modifier in {"const", ""}
    every e in D.associations  is an Edge instance with ownership in {"composition", "aggregation", "none"}
    every e in D.dependencies  is an Edge instance with ownership == "none"
    every e in D.generalizations is an Edge instance with ownership == "none"
    every e.source and e.target are non-empty strings
    every e.name is a string (may be empty)
    no duplicate (source,target,relationship_type,name,ownership) within same list
```


```
isValidUCG(G):
    G is a networkx.MultiDiGraph
    every node has attribute 'tag' in {"vc", "va"}
    every edge has attribute 'tag' in {"e_a", "e_1", "e_2", "e_3", "e_4", "e_5"}
    no dangling edges (every edge endpoint is existing node)
```


```
isValidSimilarity(s):
    s is a float or int
    0.0 <= s <= 1.0
```

## Decompose functions


```
diagram = normalize(model):
   classes = []
   for cls in model.classes:
       attrs = [ AttributeInfo(a.name, (a.type or "").lower(), "const" if a.is_constant else "")
                 for a in cls.attributes ]
       classes.append(ClassInfo(cls.name, cls.is_abstract, attrs))
       for enum in cls.nested_enums:
           classes.append(enumToClass(enum))
   for enum in model.enums:
       classes.append(enumToClass(enum))
   for name in model.implicit_classes:
       classes.append(ClassInfo(name, False, []))

   associations, dependencies, generalizations = [], [], []
   for rel in model.relationships:
       for e, kind in toEdges(rel):
           if   kind == "association":   associations.append(e)
           elif kind == "dependency":    dependencies.append(e)
           elif kind == "generalization": generalizations.append(e)

   return Diagram(classes, associations, dependencies, generalizations)

   requires:
       isValidParsedModel(model)
   ensures:
       isValidDiagram(diagram)

Input:  One ParsedModel instance.
Output: A Diagram containing normalized ClassInfo, Edge lists (associations,
        dependencies, generalizations) with enums-as-classes and association-class
        expansion applied.
```


```
enumClass = enumToClass(enum):
   attrs = [ AttributeInfo(v, "enum_value", "const") for v in enum.values ]
   return ClassInfo(enum.name, False, attrs)

   requires:
       enum is instance of Parser.models.ParsedEnum
   ensures:
       result is ClassInfo instance with is_abstract == False
```


```
edges = toEdges(rel):
   t = rel.relationship_type
   name = rel.label or ""
   if t == RelationshipType.INHERITANCE:
       return [(Edge(rel.source, rel.target, name, "none"), "generalization")]
   if t == RelationshipType.DEPENDENCY:
       return [(Edge(rel.source, rel.target, name, "none"), "dependency")]
   if t == RelationshipType.COMPOSITION:
       return [(Edge(rel.source, rel.target, name, "composition"), "association")]
   if t == RelationshipType.AGGREGATION:
       return [(Edge(rel.source, rel.target, name, "aggregation"), "association")]
   if t in (RelationshipType.ASSOCIATION, RelationshipType.DIRECTED_ASSOCIATION):
       return [(Edge(rel.source, rel.target, name, "none"), "association")]
   if t == RelationshipType.ASSOCIATION_CLASS:
       x, y = rel.association_members
       z = rel.target
       return [
           (Edge(x, z, name, "none"), "association"),
           (Edge(y, z, name, "none"), "association"),
       ]
   return []

   requires:
       rel is instance of Parser.models.ParsedRelationship
   ensures:
       every item in edges is a tuple (Edge, str label)
       label in {"association", "dependency", "generalization"}
```

```
prop = propSim(d1, d2):
   C1 = d1.classes
   C2 = d2.classes
   if len(C1) == 0 and len(C2) == 0: return 1.0
   if len(C1) == 0 or  len(C2) == 0: return 0.0
   M = [[ cSim(a, b) for b in C2] for a in C1]
   bestSum = greedyOptimalSum(M)
   return (2 * bestSum) / (len(C1) + len(C2))

   requires:
       isValidDiagram(d1)
       isValidDiagram(d2)
   ensures:
       isValidSimilarity(prop)

Input:  Two Diagram instances.
Output: Aggregate class property similarity using optimal bipartite matching.
```


```
rel = relSim(d1, d2):
   RA1, RD1, RG1 = d1.associations, d1.dependencies, d1.generalizations
   RA2, RD2, RG2 = d2.associations, d2.dependencies, d2.generalizations
   N = len(RA1)+len(RD1)+len(RG1)+len(RA2)+len(RD2)+len(RG2)
   if N == 0:
       return 1.0 if all empty else 0.0
   w_ra = (len(RA1)+len(RA2)) / N
   w_rd = (len(RD1)+len(RD2)) / N
   w_rg = (len(RG1)+len(RG2)) / N
   return w_ra*raSim(RA1,RA2) + w_rd*rdSim(RD1,RD2) + w_rg*rgSim(RG1,RG2)

   requires:
       isValidDiagram(d1)
       isValidDiagram(d2)
   ensures:
       isValidSimilarity(rel)

Input:  Two Diagram instances.
Output: Aggregate relationship similarity weighted by diagram relationship counts.
```

```
struc = classStruc(d1, d2):
   g1 = buildUCG(d1)
   g2 = buildUCG(d2)
   intra = intraSim(g1, g2)
   inter = interSim(g1, g2)
   return (1 - 0.1) * intra + 0.1 * inter

   requires:
       isValidDiagram(d1)
       isValidDiagram(d2)
       NetworkX graph_edit_distance available
   ensures:
       isValidSimilarity(struc)

Input:  Two Diagram instances.
Output: A float in [0.0, 1.0] representing combined structural similarity.
```

```
graph = buildUCG(diagram):
   G = nx.MultiDiGraph()
   for c in diagram.classes:
       G.add_node(("vc", c.name), tag="vc")
       for a in c.attributes:
           attr_id = ("va", c.name, a.name)
           G.add_node(attr_id, tag="va")
           G.add_edge(("vc", c.name), attr_id, tag="e_a")
   for rel in diagram.associations:
       tag = _tagForAssociation(rel)
       G.add_edge(("vc", rel.source), ("vc", rel.target), tag=tag)
   for rel in diagram.dependencies:
       G.add_edge(("vc", rel.source), ("vc", rel.target), tag="e_5")
   for rel in diagram.generalizations:
       G.add_edge(("vc", rel.source), ("vc", rel.target), tag="e_2")
   return G

   requires:
       isValidDiagram(diagram)
   ensures:
       isValidUCG(graph)

Input:  One Diagram instance.
Output: A MultiDiGraph representing the UML Common Graph with typed vertices
        and edges (vc, va, e_a, e_1..e_5).
```

```
intra = intraSim(g1, g2):
   sg1 = intraSubgraphs(g1)
   sg2 = intraSubgraphs(g2)
   if not sg1 and not sg2: return 1.0
   if not sg1 or  not sg2: return 0.0
   M = [[ gedNormalized(s1, s2) for s2 in sg2] for s1 in sg1]
   bestSum = greedyOptimalSum(M)
   return (2 * bestSum) / (len(sg1) + len(sg2))

   requires:
       isValidUCG(g1)
       isValidUCG(g2)
   ensures:
       isValidSimilarity(intra)

Input:  Two UCG MultiDiGraphs.
Output: Normalized Graph-Edit-Distance similarity over per-class intra-subgraphs
        using greedy optimal bipartite matching.
```

## Black-box Dependencies


```
CosineSim(s1, s2) -> float:
   Black-box NLP primitive.
   Implements camelCase-aware tokenisation, POS-tagging, stopword removal,
   lemmatisation, semantic-vocabulary filtering (tau = 0.85), and cosine
   similarity between weighted WordNet vectors.

   requires:
       s1, s2 are strings
       isLoadedWordNet()
   ensures:
       isValidSimilarity(result)
```


```
greedyOptimalSum(M) -> float:
   Implements the greedy algorithm:
   - Repeatedly select the maximum value in matrix M
   - Zero its entire row and column
   - Sum selected values until no rows/columns remain
   Returns the sum.

   requires:
       M is a 2-D numeric matrix (list-of-lists or numpy ndarray)
   ensures:
       result is a non-negative real number
```


```
gedNormalized(g1, g2) -> float:
   Computes graph edit distance between two networkx graphs, normalised by the
   larger graph's combined node+edge count.
   cost = graph_edit_distance(g1, g2, node_match, edge_match)   # tag equality
   size = max(|nodes(g1)|+|edges(g1)|, |nodes(g2)|+|edges(g2)|)
   if size == 0: return 1.0
   else: return 1 - cost/size

   requires:
       g1, g2 are networkx graphs
       every node/edge has a 'tag' attribute for matching
       isLoadedNetworkX()
   ensures:
       isValidSimilarity(result)
```


```
clampToUnit(x) -> float:
   returns max(0.0, min(1.0, float(x)))

   ensures:
       isValidSimilarity(result)
```
