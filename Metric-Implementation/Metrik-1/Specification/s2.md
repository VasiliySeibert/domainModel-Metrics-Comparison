# I-0 Specification

```
metric(instructor_model, student_model):
   mapping |= mapClasses(instructor_model, student_model)
   mapping |= mapRelations(instructor_model, student_model, mapping)
   mistakes  = checkClasses(mapping)
   mistakes += checkRelations(mapping)
   mistakes += checkMissing(mapping)
   result = normalize(mistakes, instructor_model, student_model)
   return result

   requires:
       isValidModel(instructor_model),
       isValidModel(student_model)
   ensures:
       isValidNormalize(result)

Input:  a parsed domain model
Output: A MetricResult dict with class_score, attribute_score, association_score in [0.0, 1.0].
```

## Invariants


```
isValidNormalize(result):
    1. result is a dict (MetricResult)
    2. result contains exactly three keys: class_score, attribute_score, association_score
    3. Every value is a float or int in the range [0.0, 1.0]
```

```
isValidModel(M):
    checks if model adheres to the data structure.
    class names are unique within M
    enumeration names are unique within M
    every relation references classes that exist in M
    every attribute belongs to exactly one class in M
```


```
isValidMapping(mapping):
    1. ParsedMapping and ClassMapping are valid instances
    2. All items in mapped_classes are MappedClass instances
    3. No student class is mapped twice
    4. No instructor class is mapped twice (prevents double-counting)
    5. No empty or None class names in mappings
    6. No duplicates in unmapped_instructor_classes or unmapped_student_classes
    7. No overlap between mapped and unmapped class sets
    8. Every MappedAttribute.student_attr is non-empty
    9. RelationshipMapping is a valid instance
    10. All items in mapped_relationships are MappedRelationship instances
    11. No student_rel_index mapped twice; no instructor_rel_index mapped twice
    12. No negative indices in mapped_relationships
    13. No duplicates in unmapped_instructor_relationships / unmapped_student_relationships
    14. No overlap between mapped and unmapped relationship index sets
```


```
isValidMistakes(mistakes):
    1. Input is a list
    2. Every item is a ParsedMistake instance
    3. Every mistake_id is in the KNOWN_MISTAKE_IDS set
    4. Every description is a non-empty string (after strip())
    5. No duplicate mistake_ids in the list
```
## Decompose functions

```
mapping = mapClasses(instructor_model, student_model):
   unmatched_ic, unmatched_sc = instructor_model.classes, student_model.classes
   mapping  = greedyNameMatch(unmatched_ic, unmatched_sc)
   mapping += greedyAttributeMatch(unmatched_ic, unmatched_sc)
   return mapping

   requires:
       isValidModel
   ensures:
       isValidMapping

Input: Two ParsedModel instances (instructor and student), each containing a list of classes with unique names and attributes.
Output: A ParsedMapping whose ClassMapping contains one-to-one MappedClass entries for every matched pair, plus unmapped_instructor_classes and unmapped_student_classes for the leftovers.
```

```
mapping |= mapRelations(instructor_model, student_model, mapping):
   unmatched_ir, unmatched_sr = instructor_model.relationships, student_model.relationships
   mapping += greedyRelMatch(unmatched_ir, unmatched_sr, mapping)
   mapping += greedyGenMatch(unmatched_ir, unmatched_sr, mapping)
   return mapping

   requires:
       isValidMapping
   ensures:
       isValidMapping

Input:  Two ParsedModel instances and a ParsedMapping produced by mapClasses.
        Association-class relationships are ignored.
Output: An updated ParsedMapping whose RelationshipMapping contains mapped
        relationship pairs, plus unmapped_instructor_relationships and
        unmapped_student_relationships indices for the leftovers.
```

```
mistakes = checkClasses(mapping, instructor_model, student_model):
   for each mapped class pair:
       if mapping_type == RENAME:        emit mistake 12
       for each missing instructor attribute: emit mistake 3
       for each extra student attribute:      emit mistake 4
       for each renamed attribute:          emit mistake 13
       for each wrong attribute type:       emit mistake 5
   return mistakes

   requires:
       isValidMapping(mapping)
       isValidModel(instructor_model)
       isValidModel(student_model)
   ensures:
       isValidMistakes(mistakes)

Input:  A ParsedMapping and the two original ParsedModel instances.
Output: A list of ParsedMistake objects for class- and attribute-level issues.
```

```
mistakes = checkRelations(mapping, instructor_model, student_model):
   for each mapped relationship pair:
       if mapping_type == TYPE_CHANGE:     emit mistake 14
       if is_inverted == True:             emit mistake 9
       if cardinalities differ:             emit mistake 8
   return mistakes

   requires:
       isValidMapping(mapping)
       isValidModel(instructor_model)
       isValidModel(student_model)
   ensures:
       isValidMistakes(mistakes)

Input:  A ParsedMapping and the two original ParsedModel instances.
Output: A list of ParsedMistake objects for relationship-level issues.
```

```
mistakes = checkMissing(mapping, instructor_model, student_model):
   for each unmapped instructor class:          emit mistake 1
   for each unmapped student class:              emit mistake 2
   for each unmapped instructor relationship:
       if INHERITANCE:       emit mistake 10
       if COMPOSITION/AGG:   emit mistake 15
       else:                 emit mistake 6
   for each unmapped student relationship:
       if INHERITANCE:       emit mistake 11
       if COMPOSITION/AGG:   emit mistake 16
       else:                 emit mistake 7
   return mistakes

   requires:
       isValidMapping(mapping)
       isValidModel(instructor_model)
       isValidModel(student_model)
   ensures:
       isValidMistakes(mistakes)

Input:  A ParsedMapping and the two original ParsedModel instances.
Output: A list of ParsedMistake objects for missing/extra elements.
```

```
result = normalize(mistakes, instructor_model, student_model)
   class_mistakes   = count mistakes in {1, 2, 12}
   attr_mistakes    = count mistakes in {3, 4, 5, 13}
   assoc_mistakes   = count mistakes in {6, 7, 8, 9, 10, 11, 14, 15, 16}
   denom_classes    = max(1, len(instructor_model.classes))
   denom_attrs      = max(1, sum(len(c.attributes) for c in instructor_model.classes))
   denom_assocs     = max(1, len(instructor_model.relationships))
   class_score      = clamp(1.0 - class_mistakes / denom_classes, 0.0, 1.0)
   attribute_score  = clamp(1.0 - attr_mistakes  / denom_attrs,   0.0, 1.0)
   association_score= clamp(1.0 - assoc_mistakes / denom_assocs,  0.0, 1.0)
   return MetricResult(class_score, attribute_score, association_score)

   requires:
       isValidMistakes(mistakes)
       isValidModel(instructor_model)
       isValidModel(student_model)
   ensures:
       isValidNormalize(result)

Input:  A validated list of ParsedMistake objects plus the two original ParsedModel
        instances.
Output: A MetricResult dict with three keys (class_score, attribute_score,
        association_score), each a float in [0.0, 1.0]. The instructor model
        provides the denominators (element counts); the student model is available
        for symmetry but not used in the denominator calculation.
```
